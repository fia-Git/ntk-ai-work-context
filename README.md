# ntk-ai-work-context

このリポジトリは、NTKCuttingToolsでの業務において、AIに自分の仕事内容・文章の好み・業務上の文脈を理解してもらい、資料作成や業務相談を効率化するための業務用コンテキスト集です。

また、自分と上司・関係者とのやり取りを記録し、相手ごとの対応方針やコミュニケーション上の注意点を整理することで、より実務に合った回答や提案をAIに行ってもらうことを目的としています。

---

## 目的

このリポジトリの主な目的は以下です。

- AIに自分の仕事内容を理解してもらう
- AIに自分の口調や文章の好みを理解してもらう
- 資料作成や報告文作成の工数を減らす
- 上司や関係者とのやり取りを整理する
- 上司や関係者ごとの対応方針を整理する
- 進行中の案件や業務内容をAIに把握してもらう
- 業務相談や仕事の進め方について、継続的な文脈を持ってAIに相談できるようにする

---

## AIに手伝ってほしいこと

AIには主に以下の業務を支援してもらいます。

- 資料作成
- 上司への報告文の作成
- チャット内容の整理
- 仕事の進め方の相談
- 会議メモの整理
- メール文面の作成
- 業務改善案の作成
- 進行中案件の整理
- ToDoや確認事項の抽出
- 相手に合わせた伝え方の整理

---

## このリポジトリに入れる情報

このリポジトリには、今後以下のような情報を整理していきます。

- 自分の口調や文章の好み
- 自分の仕事内容
- 上司とのやり取り
- ユーザーさんや関係者とのやり取り
- 進行中の案件
- 資料作成のルール
- よく使うテンプレートファイル
- 業務上の用語
- 好みのファイルや参考資料
- 相手ごとのコミュニケーション上の特徴
- AIに覚えておいてほしい業務上の前提

---

## AIへの基本指示

AIは、このリポジトリの内容を参照する際、以下の方針を守ってください。

- 基本は日本語で回答する
- 結論から先に書く
- 長文だけで説明せず、できるだけ箇条書きで整理する
- 不明点は推測で断定せず、必要に応じて確認する
- 上司向けの文章は、できるだけ5W1Hが分かる形にする
- 特に「目的」や「なぜ必要なのか」は丁寧に説明する
- 資料作成時は、実務でそのまま使える形に整える
- 報告文や相談文は、相手に失礼がない丁寧な表現にする
- 相手ごとの特徴を参考にする場合は、決めつけではなく「対応上の傾向」として扱う
- 会社情報や個人情報の取り扱いには注意する

---

## 情報の取り扱いルール

このリポジトリには業務上の情報が含まれる可能性があるため、以下のルールを守ります。

- パスワード、APIキー、トークンなどの認証情報は絶対に記載しない
- 必要以上に個人情報を記載しない
- 上司や関係者とのやり取りは、必要に応じて要約して記録する
- 相手の特徴を書く場合は、感情的な評価ではなく、業務上の対応に役立つ事実や傾向として整理する
- チームメンバーに共有する可能性があることを前提に、見られて困る内容は書かない
- AIに読み込ませる前提で、誤解を招く表現や断定的な人物評価は避ける
